# Instrumentation Calibration Workshop

Boiler-plate and solution code for Rocket Project Instrumentation Calibration Workshop on 11/10

See the [installation guide](https://docs.google.com/document/d/1vfM4OlHG4ca5h7T67_1kjNCmjwNy4su1gDDBjYXepoY/edit?tab=t.0) for setup instructions.

- See ``/pt_reader.ino`` for the script we will flash onto the ESP32 on the DAQ.
- See ``/cal.ipynb`` for the Jupyter Notebook / Python script we will do a Linear Regression on in [Google Colab](https://colab.research.google.com/)
